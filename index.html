Game Rules

  1. Game Board: The game board consists of a grid of squares. Each square may contain a mine or be empty.

  2. Mines: mines are hidden within the grid squares. The number of mines and the size of the board can vary depending on the difficulty level chosen by the player.

  3. Opening Squares: The game begins with the player selecting a square on the grid to open. If the selected square contains a mine, the game is over, and the mine detonates. If the square is empty, the game continues, and the square reveals a number indicating how many mines are adjacent to it (including diagonals).

  4. Numbers: Each number revealed on the board represents the count of mines adjacent to that square. For example, if a square reveals the number "3," it means that there are three mines in the eight squares surrounding it.

  5. Empty Squares: If a square is revealed to be empty (i.e., it has no adjacent mines), it automatically opens all adjacent empty squares.

  6. Flagging Mines: Players can mark squares they suspect contain mines by flagging them. This helps to keep track of where the mines might be.

  7. Winning: The player wins the game when all non-mine squares are opened. This means that all mines have been correctly flagged, and all non-mine squares have been revealed.

  8. Losing: The player loses the game if they reveal a square containing a mine. In this case, all mines on the board are typically revealed, and the game ends.
